  <action>
Create test/conformance/helpers_test.go in package conformance_test with shared helpers extracted from the repeated boilerplate in dataout_test.go. The helpers must cover:

1. **writeTestSetup** struct and constructor: Encapsulates the common pattern of creating MockTarget + Recorder + configuring negotiation + registering HandleLogin/HandleLogout/HandleNOPOut. Signature:
   ```go
   type writeTestSetup struct {
       Target   *testutil.MockTarget
       Recorder *pducapture.Recorder
   }
   
   func newWriteTestSetup(t *testing.T, negCfg testutil.NegotiationConfig) *writeTestSetup
   ```
   The constructor calls NewMockTarget, registers t.Cleanup for Close, sets negotiation config, registers HandleLogin/HandleLogout/HandleNOPOut, creates a Recorder. Returns the setup struct.

2. **dialWithOverrides** helper: Dials the MockTarget with PDU hook, keepalive=30s, and the given operational overrides map. Returns *uiscsi.Session. Signature:
   ```go
   func (s *writeTestSetup) dialWithOverrides(t *testing.T, ctx context.Context, overrides map[string]string) *uiscsi.Session
   ```
   Calls uiscsi.Dial with s.Target.Addr(), WithPDUHook(s.Recorder.Hook()), WithKeepaliveInterval(30*time.Second), WithOperationalOverrides(overrides). Registers t.Cleanup for sess.Close. Fatals on error.

3. **negotiationOverrides** helper: Converts NegotiationConfig booleans to the string map format for WithOperationalOverrides. Signature:
   ```go
   func negotiationOverrides(cfg testutil.NegotiationConfig) map[string]string
   ```
   Maps ImmediateData *bool to "Yes"/"No", InitialR2T *bool to "Yes"/"No". Omits nil fields. This eliminates the manual string map construction that is duplicated in every test.

4. **sendR2TAndConsume** helper: Common target-side handler pattern for write tests -- sends an R2T for a given offset/length and reads Data-Out until F-bit. Signature:
   ```go
   func sendR2TAndConsume(tc *testutil.TargetConn, tgt *testutil.MockTarget, cmd *pdu.SCSICommand, ttt uint32, offset uint32, length uint32) (expCmdSN, maxCmdSN uint32, err error)
   ```
   Calls tgt.Session().Update, sends R2T, calls ReadDataOutPDUs, returns expCmdSN/maxCmdSN for the response.

5. **sendSCSIResponse** helper: Sends a standard success SCSI Response. Signature:
   ```go
   func sendSCSIResponse(tc *testutil.TargetConn, cmd *pdu.SCSICommand, expCmdSN, maxCmdSN uint32) error
   ```

All helpers use t.Helper() where appropriate. Import paths: github.com/rkujawa/uiscsi, github.com/rkujawa/uiscsi/internal/pdu, github.com/rkujawa/uiscsi/test (as testutil), github.com/rkujawa/uiscsi/test/pducapture.

Do NOT refactor existing dataout_test.go to use these helpers -- that is out of scope. The helpers are designed so new tests (scsicommand_test.go) can use them, and existing tests can optionally adopt them later.
