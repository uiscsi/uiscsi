**TestSNACK_DataSNGap** (SNACK-01, FFP #13.1):
Verifies the initiator sends a Data/R2T SNACK (Type=0) with correct BegRun and RunLength when a DataSN gap is detected.

Setup:
1. Create Recorder.
2. Create MockTarget with ERL=1: `tgt.SetNegotiationConfig(testutil.NegotiationConfig{ErrorRecoveryLevel: testutil.Uint32Ptr(1)})`.
3. Register HandleLogin, HandleLogout, HandleNOPOut.

Handler (HandleSCSIFunc):
- Use `tgt.Session().Update(cmd.CmdSN, cmd.Header.Immediate)` for CmdSN tracking.
- Prepare 1536 bytes of test data (3 x 512-byte PDUs).
- Send DataSN=0 (512 bytes at offset 0, Final=false).
- SKIP DataSN=1 (creating the gap).
- Send DataSN=2 (512 bytes at offset 1024, Final=false).
- Wait 100ms for SNACK to propagate.
- Retransmit DataSN=1 (512 bytes at offset 512, Final=false) -- the data the gap requested.
- Send final DataSN=3 with HasStatus=true, Status=0x00, Final=true, DataSegmentLen=0.

Dial with: WithPDUHook(rec.Hook()), WithKeepaliveInterval(30*time.Second), WithOperationalOverrides(map[string]string{"ErrorRecoveryLevel": "1"}).

Test assertions:
- ReadBlocks should succeed (gap recovered via SNACK + retransmit).
- time.Sleep(100ms) after ReadBlocks.
- `snacks := rec.Sent(pdu.OpSNACKReq)` -- must have at least 1 SNACK.
- Find SNACK with Type==0 (Data/R2T SNACK).
- Verify BegRun==1 (the missing DataSN).
- Verify RunLength==1 (one missing PDU).
- Verify TargetTransferTag is set (echoes TTT from the task context -- value depends on implementation, just verify it's present).

Use 10-second context timeout. Use t.Cleanup for cleanup.

Imports needed: context, testing, time, pdu, testutil (test package), pducapture, uiscsi.
