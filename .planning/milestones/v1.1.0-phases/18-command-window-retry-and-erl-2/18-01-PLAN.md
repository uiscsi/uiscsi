Create test/conformance/cmdwindow_test.go with package conformance_test.

**TestCmdWindow_ZeroWindow (CMDSEQ-04, per D-03 and D-04):**
- Create MockTarget with HandleLogin, HandleLogout, HandleNOPOut.
- Attach pducapture.Recorder via WithPDUHook.
- HandleSCSIFunc: on callCount==0, call tgt.Session().SetMaxCmdSNDelta(-1) BEFORE Update(), then send normal SCSI Response with correct ExpCmdSN/MaxCmdSN (window closed). After sending the response, launch a goroutine that sleeps 500ms, then calls SetMaxCmdSNDelta(10) and sends an unsolicited NOP-In (ITT=0xFFFFFFFF, TTT=0xFFFFFFFF) with the reopened ExpCmdSN/MaxCmdSN using tc.NextStatSN(). On callCount>=1, respond normally with SetMaxCmdSNDelta(10).
- Dial with WithKeepaliveInterval(30*time.Second) to suppress keepalive NOP-Outs.
- First ReadBlocks: succeeds (closes window in response).
- Launch goroutine for second ReadBlocks into a done channel.
- Use select with time.After(300ms) to verify the goroutine does NOT return (proves blocking). If it returns, t.Fatal.
- Use select with time.After(5s) to verify the goroutine DOES complete after NOP-In reopens window. If timeout, t.Fatal.
- Verify via pducapture that exactly 2 SCSI Command PDUs were sent, and the second one has CmdSN > first CmdSN.

**TestCmdWindow_LargeWindow (CMDSEQ-05, per D-03):**
- Create MockTarget with HandleLogin, HandleLogout, HandleNOPOut.
- Attach pducapture.Recorder.
- HandleSCSIFunc: always call SetMaxCmdSNDelta(255) before Update(), respond with Data-In (HasStatus=true, Status=0x00) containing 512 bytes.
- Dial session.
- Launch 8 goroutines each calling ReadBlocks(ctx, 0, lba, 1, 512) with different LBAs (0-7).
- Collect all results via errgroup or channels. All must succeed.
- Verify pducapture shows all 8 SCSI Command PDUs sent, each with unique CmdSN values monotonically increasing.

**TestCmdWindow_MaxCmdSNInResponse (CMDSEQ-09, per D-03):**

- Dial session.
- First ReadBlocks succeeds (window open).
- Second ReadBlocks succeeds (response closes window).
- Third ReadBlocks in goroutine -- verify it blocks (300ms select) then completes after NOP-In (5s select).
- Verify pducapture shows 3 SCSI Commands sent.
