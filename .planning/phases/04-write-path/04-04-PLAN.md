2. Add a new exported option function:
```go
// WithCloseTimeout sets the deadline for the graceful logout attempt in Close().
// A zero or negative duration skips the graceful logout entirely.
func WithCloseTimeout(d time.Duration) SessionOption {
	return func(c *sessionConfig) {
		c.closeTimeout = d
	}
}
```

3. In `internal/session/session.go`, modify the `Close()` method. Replace the hardcoded timeout on line 212-216:
```go
if wasLoggedIn && s.err == nil {
    ctx, cancel := context.WithTimeout(context.Background(), 5*time.Second)
    _ = s.logout(ctx, 0)
    cancel()
}
```
with:
```go
if wasLoggedIn && s.err == nil && s.cfg.closeTimeout > 0 {
    ctx, cancel := context.WithTimeout(context.Background(), s.cfg.closeTimeout)
    _ = s.logout(ctx, 0)
    cancel()
}
```

The key change is: (a) use `s.cfg.closeTimeout` instead of `5*time.Second`, and (b) add `s.cfg.closeTimeout > 0` guard so zero timeout skips logout entirely without creating a context.
